# 简短安装说明

本说明仅适用于 `Cursor`。

如果你要安装 `Codex CLI` 版本，请返回上级目录并使用 `codex一键安装技能/` 下的说明和脚本。

## Windows
1. 双击 `一键安装最新版Cursor技能和MCP.bat`
2. 重启 Cursor
3. 如需确认安装成功，双击 `一键自检Cursor技能和MCP.bat`
4. `ui-ux-pro-max` 会一起安装；运行该技能需要 `python3` 或 `python`

## macOS / Linux
1. 在终端进入当前目录
2. 运行：

```bash
chmod +x install_cursor_skills.sh repair_cursor_skills.sh self_check_cursor_skills.sh
./install_cursor_skills.sh
```

3. 重启 Cursor
4. 如需确认安装成功，运行：

```bash
./self_check_cursor_skills.sh
```

5. `ui-ux-pro-max` 会一起安装；运行该技能需要 `python3` 或 `python`

## 关键路径

- 技能目录：`~/.cursor/skills/`
- MCP 配置：`~/.cursor/mcp.json`
- Context7 本地运行时：`~/.cursor/context7-local/`
- Context7 启动器：`~/.cursor/run-context7.sh` 或 `run-context7.cmd`

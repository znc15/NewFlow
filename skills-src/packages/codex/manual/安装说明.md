# 纯手动安装说明

本说明仅适用于 `Codex CLI`。

如果你要安装 `Cursor` 版本，请改用 `../../cursor一键安装技能/纯手动安装/` 下的文档和脚本。

## 一、手动复制技能
把当前目录下的 `skills` 整个复制到目标机器的 Codex 技能目录：

- 优先：`$CODEX_HOME/skills/`
- Windows 默认：`%USERPROFILE%\.codex\skills\`
- macOS/Linux 默认：`~/.codex/skills/`

复制后的结构应类似：

```text
~/.codex/skills/brainstorming/
~/.codex/skills/playwright/
~/.codex/skills/code-review/
```

复制完成后，重启 Codex CLI。

## 二、安装 context7 MCP
如果你希望 `context7` 也能使用，有两种办法。

### 办法 A：手动改 config.toml
模板文件在：

- `context7-config-templates/config.local.windows.toml`
- `context7-config-templates/config.local.unix.toml`
- `context7-config-templates/config.remote.toml`

说明：

- `config.local.windows.toml` / `config.local.unix.toml` 走本地 launcher + bundled runtime
- `config.remote.toml` 走远程 MCP
- 默认优先使用 bundled runtime；`config.remote.toml` 只作为高级手动选项保留

### 办法 B：运行这里自带的 context7 专用脚本

- Windows：双击 `一键安装context7-MCP.bat`
- macOS/Linux：

```bash
chmod +x install_context7_mcp.sh repair_context7_mcp.sh
./install_context7_mcp.sh
./install_context7_mcp.sh --target-codex-home /path/to/.codex
```

脚本会自动：

- 检查 `codex` 和 `node` 是否可用
- 复制 bundled `context7-local` 到 `CODEX_HOME/context7-local`
- 生成 `run-context7.cmd` 或 `run-context7.sh`
- 写入或修复 `config.toml`

## 三、playwright 说明
`playwright` 技能手动复制后会被 Codex 识别，但运行相关命令仍要求目标机器具备：

```text
node
npm
npx
```

## 四、ui-ux-pro-max 说明
`ui-ux-pro-max` 已经包含在 bundled skills 中。复制到 `CODEX_HOME/skills/` 后，Codex 能识别这个技能，但运行它的搜索脚本仍要求目标机器具备：

```text
python3 或 python
```

如果你要刷新 `ui-ux-pro-max`，请使用仓库根目录的更新脚本：

```text
一键更新ui-ux-pro-max技能.bat
update-ui-ux-pro-max-skill.ps1
update-ui-ux-pro-max-skill.sh
```

这些脚本不会联网；它们只是把仓库里内置的 `ui-ux-pro-max` 重新复制到你的 `CODEX_HOME/skills/ui-ux-pro-max`。

如果你只在 `纯手动安装/` 目录里操作，也可以直接运行本目录里的独立脚本：

```text
install-ui-ux-pro-max-skill.ps1
repair-ui-ux-pro-max-skill.ps1
一键安装ui-ux-pro-max技能.bat
一键修复ui-ux-pro-max技能.bat
install_ui_ux_pro_max_skill.sh
repair_ui_ux_pro_max_skill.sh
```

## 五、验证
安装后可在 Codex 中测试：

```text
Use `code-review` to review the changed files in this workspace and list findings first.
```

```text
Use `playwright` to automate a simple browser flow and explain the run steps.
```

```text
Use context7 to resolve the official React library ID.
```

## 六、关键边界

- 技能目录：`$CODEX_HOME/skills/` 或 `~/.codex/skills/`
- MCP 配置：`$CODEX_HOME/config.toml` 或 `~/.codex/config.toml`
- Context7 启动器：`run-context7.cmd` / `run-context7.sh`
- 安装后重启目标：`Codex CLI`

# context7 配置模板说明

## 本地 stdio 方式
- Windows：`config.local.windows.toml`
- macOS/Linux：`config.local.unix.toml`

本地方式要求目标机器具备：
- `node`
- 已安装到本地的 `context7-local` 运行时
- 对应平台的 launcher：
  - Windows：`run-context7.cmd`
  - macOS/Linux：`run-context7.sh`

## 远程 HTTP 方式
- `config.remote.toml`

远程方式不依赖本地运行时，但前提是你的 Codex CLI 版本支持远程 MCP `url` 配置，并且目标环境可访问 `https://mcp.context7.com/mcp`。

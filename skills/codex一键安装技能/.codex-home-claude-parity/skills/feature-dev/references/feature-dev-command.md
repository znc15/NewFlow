# Codex-Native Feature Development Command

Use this workflow when a feature request needs structured discovery before implementation.

## Workflow

1. Capture the request and create an update_plan.
2. Explore the codebase with focused research passes.
3. Ask clarifying questions before design.
4. Compare architectural approaches and recommend one.
5. Wait for user approval before implementation.
6. Implement the approved design with focused changes.
7. Run review passes and summarize the final result.
